# web-project

## Node.js

아래 링크로 들어간 다음 OS를 선택하고 다운받고 설치

[nodejs](https://nodejs.org/ko/download/current/)

설치가 완료 되었으면 nodejs와 npm이 잘 설치가 되었는지 확인해보자

> nodejs version

```
node -v
```

> npm version

```
npm -v
```

## Yarn

npm이 잘 설치되었다면 yarn을 설치해보자

> install

```
npm install -g yarn
```

> yarn version

```
yarn -v
```

이상, 모든 설치가 완료되었다면 다음으로 넘어간다

## Project setup

### 의존성 설치

프로젝트를 원활히 실행하기 위해 필요한 모듈을 설치한다

```
yarn install
```

### 컴파일 및 개발 서버 실행

```
yarn serve
```
